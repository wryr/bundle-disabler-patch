# bundle-disabler-patch
Here is a simple, single purpose, light weight addon with no constant code running, for completely disabling all bundle functionality of every color.

### Features:
> - Doesn't conflict with any other addon.
> - Easy apply & instantly working.
> - Lightweight, no constant code running.
> - Overwrites vanilla bundle functionality.
> - Disables ALL bundles functionality.
> - Prevents & patches the bundle dupe glitch.
> - No back door & no obfuscation, created by ARAS Staff (me).
> - Will not break in future Minecraft updates.
> - Doesn't require resource pack.


Simply download the mcpack file below and apply it to your world/realm!
